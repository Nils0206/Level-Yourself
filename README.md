# Level-Yourself
-> Design:
  - Fix Window Größe
  - Dunkel Gehalten, Weiß umrandet
  - Rechteckig
-> Oberkategorien:
    - Namen / Art
    - Allgemeines Level
    - Progressbar
    - Emoji
-> Unterkategorien:
    - Namen
    - Level
    - Progressbar
    - Emoji

-> Funktion:
  -> Main Window
   - Hinzufügen von Oberkategorien
   - Anpassen der Obekategorien
       - Namen anpassen
       - Emoji ändern
       - Text farbe ändern
-> Oberkategorie
       - Hinzufügen von Unterkategorien
       - Rechner / Festlegung für die Level
-> Unterkategorie
       - Rechnung / Festlegung für Level
       - Namen ändern
       - Textfarbe ändern
       - Emoji ändern
